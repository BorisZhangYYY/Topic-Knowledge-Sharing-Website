# v0.5 测试日志

## 1. 问题一览

1. Register页面，点击Register按钮后没有跳转回登录页，且没有注册邮箱相关选项；
2. 如果选择忘记密码，会向响应数据中发验证码，但是由于用户没有注册邮箱实际上，导致无法重置密码；
3. 当前用户比如成功登录后，再次切换回“/”还是需要重新登录，意思是这些登录信息没有被保存，需要修复。

## 2. 前端优化
1. 首先，所有界面右下角的那个设置背景主题，比较难看，需要优化，比如可以支持自定义上传图片，为了保证永久性，可以落入保存以便加载后续；
2. 其次，这个按钮在点击其他地方的时候不会自动消失，总之这种悬浮的情况，在你点击其他地方时都应该修改让其消失的；
